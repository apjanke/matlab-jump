org.apache.pdfbox.rendering

Class PageDrawer

  • java.lang.Object
    • org.apache.pdfbox.contentstream.PDFStreamEngine
      • org.apache.pdfbox.contentstream.PDFGraphicsStreamEngine
        • org.apache.pdfbox.rendering.PageDrawer

  • public class PageDrawer
extends PDFGraphicsStreamEngine
    Paints a page in a PDF document to a Graphics context. May be subclassed to provide custom rendering.

    If you want to do custom graphics processing rather than Graphics2D rendering, then you should subclass PDFGraphicsStreamEngine instead. Subclassing PageDrawer is only suitable for cases where the goal is to render onto a Graphics2D surface.
